# UNP_PROJECT_E_COMMERCE_DATA_ANALYSIS_USING_MySQL


I did this project in collaboration with UNP, where I analyzed e-commerce data using MySQL to extract meaningful business insights.
The dataset includes customers, orders, products, and sales details. Using SQL queries like SELECT, JOIN, GROUP BY, and aggregate functions, I analyzed sales performance, customer behavior, and product trends.
This project helped in identifying top-selling products, high-value customers, and overall revenue patterns, supporting better business decision-making.
